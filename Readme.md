# Priscilla Beach Association WordPress Site

Custom code and project documentation for the Priscilla Beach Association WordPress site.

This repository is intended to track **our custom code only**, not the entire WordPress runtime.

## Repository Purpose

This repo stores:

- custom WordPress theme code
- custom PBA plugins
- project documentation
- deployment notes
- local development notes

This repo does **not** store:

- WordPress core
- Local app configuration
- logs
- uploads/media
- cache files
- backups
- third-party plugins unless explicitly included

## Local Site Structure

The Local site root is:

```text
priscilla-beach-association/
  app/
  conf/
  logs/

```
